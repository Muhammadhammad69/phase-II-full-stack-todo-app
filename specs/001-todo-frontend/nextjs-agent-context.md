# Agent Context Update

## Next.js Development Agent Context

The following technologies and patterns are used in this project:

### Core Technologies
- Next.js 16+ with App Router
- TypeScript with strict typing
- React Server Components and Client Components
- CSS Modules for styling
- Tech Innovation theme from theme-factory

### Key Patterns
- Server Components for data fetching and static content
- Client Components for interactive elements
- Component-based architecture with clear separation
- TypeScript interfaces aligned with backend models
- Responsive design with mobile-first approach

### Theme Implementation
- Electric Blue (#0066ff) as primary color
- Neon Cyan (#00ffff) as highlight color
- Dark Gray (#1e1e1e) as background
- DejaVu Sans typography

### Component Structure
- UI components in components/ui/ (Button, Input, Modal, etc.)
- Feature components in components/todo/ (TaskItem, TaskList, etc.)
- Types defined in app/types/ and aligned with backend models
