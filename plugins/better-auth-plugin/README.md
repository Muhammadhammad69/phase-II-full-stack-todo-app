# Better-Auth Plugin

This plugin automates Better-Auth authentication setup, configuration, and management. It helps developers quickly set up OAuth providers, manage users, follow security best practices, and resolve common authentication issues.

## Features

- Initialize Better-Auth projects with proper configuration
- Set up OAuth providers (Google, GitHub, Facebook, Discord, Twitter)
- Create and manage users
- Generate authentication configuration files
- Test complete authentication flows
- Provide security guidance and best practices
- Troubleshoot authentication errors
- Suggest performance optimizations

## Commands

- `/ba:init-auth` - Initialize Better-Auth configuration and project structure
- `/ba:setup-oauth` - Configure OAuth providers with credentials
- `/ba:create-user` - Create new users with customizable properties
- `/ba:test-auth` - Test various authentication flows
- `/ba:generate-config` - Generate configuration files in different formats
- `/ba:validate` - Validate configuration and environment variables
- `/ba:troubleshoot` - Diagnose and troubleshoot authentication issues

## Agents

- Project Initializer - Automated project setup and configuration
- Authentication Validator - Validates auth flows and configurations
- OAuth Expert - Specialized assistance with OAuth provider setup
- Security Auditor - Reviews security configurations and suggests improvements

## Skills

- Authentication Best Practices - Covers password hashing, session management, etc.
- Security Guidance - Provides security best practices and vulnerability prevention
- Troubleshooting - Addresses common Better-Auth issues and debugging
- OAuth Configuration - Guides through OAuth provider setup
- Database Configuration - Helps with database setup for authentication

## Prerequisites

- Node.js 16+
- Better-Auth installed in your project
- Appropriate OAuth provider credentials (for OAuth setup)

## Installation

This plugin works with Claude Code and integrates seamlessly with your development workflow.
